# Sideral : Elden Ring

Source repository for SIDERAL, an Elden Ring mod tool that generates a modified
`regulation.bin` to transform selected weapon rewards and loot sources into
sorceries and incantations.

This repository is provided for source review and build transparency.

## Current version

`V1.0.3`

## Source review notes

## Important repository notes

This repository intentionally does not include:

- Elden Ring game files.
- Generated `Output` files.
- Nexus Mods release archives.
- Mod Engine 2 binaries.
- Local binary dependencies under `lib`.
- Yabber binaries/tools.

Those files are either generated locally, provided by the user's local game
installation, downloaded separately, or third-party runtime dependencies.
