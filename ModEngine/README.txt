Mod Engine 2 n'est pas versionne avec le code source SIDERAL.

Pour utiliser le profil Autonome, telechargez Mod Engine 2 separement puis laissez SIDERAL vous demander de selectionner :
modengine2_launcher.exe

SIDERAL peut aussi le trouver automatiquement si vous placez Mod Engine 2 dans une structure locale compatible, par exemple :
ModEngine\modengine2_launcher.exe

Source officielle :
https://github.com/soulsmods/ModEngine2/releases
